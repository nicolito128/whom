# Whom

A tool for automatically configuring and managing a set of Podman Quadlet files.

**Requirements**:

* `go 1.24.4^`
* [systemd](https://systemd.io/)
* [podman](https://github.com/containers/podman)
* [podman-compose](https://github.com/containers/podman-compose)
* [podlet](https://github.com/containers/podlet/)

## Why 'whom'?

You can give it whatever meaning you prefer:

* [whom - pronoun, formal - by Cambridge Dictionary](https://dictionary.cambridge.org/dictionary/english/whom)
* “Whom” as an acronym: Whale Orchestra Manager
